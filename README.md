# 📝 PostSphere – Modern Blogging Web App

## 🚀 Overview

PostSphere is a full-stack blogging platform where users can create, edit, and manage posts using a rich text editor and media uploads.
It is built with modern frontend technologies and powered by Appwrite as the backend service.

---

## ✨ Features

* 🔐 User Authentication (Login / Signup)
* 📝 Create, Edit & Delete Blog Posts
* 🖋 Rich Text Editor (TinyMCE)
* 🖼 Image Upload using Appwrite Storage
* ⚡ Form Handling with React Hook Form
* 🔄 State Management using Redux Toolkit
* 🎯 Dynamic Routing (React Router)
* 📱 Responsive UI with Tailwind CSS

---

## 🛠 Tech Stack

### Frontend

* React.js
* Redux Toolkit
* React Router DOM
* React Hook Form
* TinyMCE (Rich Text Editor)
* Tailwind CSS

### Backend

* Appwrite

  * Authentication
  * Database
  * Storage

---

## 🧠 Key Concepts Used

* React Hooks (useState, useEffect, useCallback)
* Controlled Forms with React Hook Form
* API Integration with Appwrite SDK
* File Upload & Preview Handling
* Component-Based Architecture

## ⚙️ Installation & Setup

1️⃣ Clone the repository
git clone https://github.com/Sumit93500/PostSphere-Modern-Blogging-Web-App.git
cd PostSphere-Modern-Blogging-Web-App

2️⃣ Install dependencies
npm install

3️⃣ Create a `.env` file in root and add:

VITE_APPWRITE_URL=your_appwrite_url
VITE_APPWRITE_PROJECT_ID=your_project_id
VITE_APPWRITE_DATABASE_ID=your_database_id
VITE_APPWRITE_COLLECTION_ID=your_collection_id
VITE_APPWRITE_BUCKET_ID=your_bucket_id

4️⃣ Run the project
npm run dev
